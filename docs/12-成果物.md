## 12.1 開発成果物

### フロントエンド
- ✅ LIFFアプリのソースコード
  - `frontend/liff-app/index.html` - エントリーポイント
  - `frontend/liff-app/js/app.js` - LIFF初期化・認証フロー
  - `frontend/liff-app/js/api.js` - API呼び出しラッパー
  - `frontend/liff-app/js/config.js` - 設定管理
  - `frontend/liff-app/css/style.css` - スタイル定義

### バックエンド
- ✅ Lambda関数のソースコード
  - `backend/lambda/auth/` - LINE認証関数（JWT検証、ユーザー登録、セッション生成）
  - `backend/lambda/award/` - スタンプ授与関数（重複チェック、DynamoDB書き込み）
  - `backend/lambda/stamps/` - スタンプ一覧取得関数（DynamoDB読み取り）
- ✅ 共通ユーティリティ
  - `backend/common/dynamodb_utils.py` - DynamoDB操作ユーティリティ
  - `backend/common/response_utils.py` - HTTPレスポンス生成ユーティリティ

- ✅ AWSリソース設定
  - DynamoDBテーブル（UserStamps, StampMasters, Users）
  - S3バケット（画像保存用）
  - API Gateway REST API（devステージ）
  - Lambda関数（auth, award, stamps）
  - IAMロール・ポリシー
